# Supply Chain For Safe Distribution of Medicines

## Overview
This project is a Java ans MySQL based web application developed to improve the transparency and security of medicine distribution.

## Features
- User authentication
- Medicine shipment tracking
- Real-time order status updates
- Secure transaction management
- MySQL database integration

## Technologies Used
- Java
- HTML
- CSS
- MySQL

## Tools
- Eclipse IDE
- MySQL Workbench
